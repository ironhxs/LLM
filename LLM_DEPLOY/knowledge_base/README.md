## 🔄 更新知识库

### 添加新文档
1. 将新文档（.txt / .md / .pdf）放入本目录
2. 运行 `notebooks/02_rag_implementation.ipynb` 重新构建向量库
3. 重启 `rag_server.py` 服务

### 支持的文档格式
- `.txt` - 纯文本（推荐）
- `.md` - Markdown 文档
- `.pdf` - PDF 文档（需要安装 pypdf）

### 文档编写建议
- 使用 UTF-8 编码
- 结构清晰，使用标题分级
- 每个主题单独成文
- 避免过长的单个文件（建议 < 50KB）
